# Shopee Products Scraper
Scrapes shopee.com products per category and saves it to a Postgres database

### Requirements
* Python 3
* Google Chrome v87
* Selenium Chrome Driver for Chrome v87

### Technology
* Python 3
* Selenium
* Multiprocessing
* Docker
* PostgreSQL

### Setup
##### Install depedencies
```
pip install -r requirements.txt
```

##### Run the application
```
python main.py
```
